Jogo da forca

Como jogar?

  • O primeiro jogador deve escolher uma palavra.
  • O segundo jogador tem o papel de advinhar a palavra escolhida pelo primeiro.
  • Para fazer isso, é preciso escolher algumas letras, na tentativa de acertar a palavra escolhida.
  • Caso a letra escolhida não faça parte da palavra, uma parte do corpo na forca será completada.
  • Se todo o corpo for completo, o segundo jogador perderá e o primeiro será o vencedor. Caso o segundo jogador acerte a palavra, ele será o vencedor e, consequentemente, o primeiro jogador perderá.

    • Regras:

  • Deve ser escolhida uma palavra pertencente à lingua portuguesa.
  • É necessário fornecer uma dica para facilitar o processo de "advinhação".
